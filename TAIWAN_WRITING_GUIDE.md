## 核心理念

### 台灣繁體中文的核心價值

**台灣繁體中文** 不只是字體的差異,更是**文化、語氣、表達方式**的不同:

| 特質 | 台灣繁體 | 大陸簡體 |
|------|---------|---------|
| **語氣偏好** | 親切、溫暖、對話式 | 正式、學術、論述式 |
| **人際距離** | 近距離(朋友感) | 遠距離(專業感) |
| **表達風格** | 柔性、委婉、情感豐富 | 硬性、直接、邏輯為主 |
| **文化關鍵詞** | 人情味、接地氣、溫度 | 理性、嚴謹、權威 |

### 台灣讀者期待

台灣讀者期待內容具備以下特質:

1. **好讀** - 易於閱讀,不費力
2. **接地氣** - 貼近生活,實用
3. **有溫度** - 感受到人性溫暖
4. **真誠感** - 不做作,不虛假
5. **共鳴感** - "這在說我!"的感覺

## 台灣 vs 大陸中文差異

### 1. 基本用詞差異

| 類別 | 台灣 | 大陸 | 說明 |
|------|------|------|------|
| **科技** | 影片 | 視頻 | Video |
| | 軟體 | 軟件 | Software |
| | 網路 | 網絡 | Internet |
| | 行動裝置 | 移動設備 | Mobile device |
| | 資訊 | 信息 | Information |
| | 程式 | 程序 | Program |
| **媒體** | 部落格 | 博客 | Blog |
| | 臉書 | 脸书 | Facebook (台灣直接說Facebook或FB) |
| | 粉絲專頁 | 公众号 | Fan page |
| **商業** | 行銷 | 營銷 | Marketing |
| | 品牌 | 品牌 | Brand (同但用法不同) |
| | 客戶 | 客戶 | Client (台灣更常用) |
| **生活** | 計程車 | 出租車 | Taxi |
| | 捷運 | 地鐵 | MRT/Subway |
| | 便利商店 | 便利店 | Convenience store |

### 2. 代詞使用差異

**台灣習慣:**
- 大量使用 **你** (casual, friendly)
- **您** 只用於極正式場合 (客服、官方公告)
- 社群媒體幾乎100%用 **你**

**大陸習慣:**
- 更常使用 **您** (respectful)
- 正式內容多用 **您**
- B2B內容常用 **您**

**建議:**
✅ 台灣內容優先使用 **你**
✅ 只有極正式B2B或政府公告用 **您**

### 3. 語氣助詞差異

| 功能 | 台灣常用 | 說明 |
|------|---------|------|
| **疑問** | 對吧？是不是？ | 尋求共鳴 |
| **肯定** | 呢、喔、啦 | 增加親切感 |
| **強調** | 超、真的、根本 | 情感加強 |
| **柔化** | 可能、應該、大概 | 避免太武斷 |

### 4. 句式結構差異

**台灣偏好:**
```
短句 + 情感詞 + 反問句
```

**大陸偏好:**
```
長句 + 邏輯連接詞 + 論證
```

**範例對比:**

台灣版:
```
你也40歲了，還在猶豫要不要創業？
我懂你的擔心。
但數據告訴我們，40歲正是好時機，你相信嗎？
```

大陸版:
```
當你40歲時仍在猶豫是否應該創業這個問題上，
你需要了解的是，根據相關數據顯示，
40歲恰恰是創業的最佳時機。
```

## 台灣語氣特點

### 1. 親切對話式 (Conversational Warmth)

**核心原則:** 像朋友聊天,不是老師授課

❌ **避免:**
```
根據研究顯示,成功創業者的平均年齡為45歲。
此數據表明,年齡並非創業成功的阻礙因素。
```

✅ **建議:**
```
你猜成功創業者平均幾歲？
45歲!驚訝吧？
所以啊，年齡根本不是問題。
```

### 2. 共感語句 (Empathy Phrases)

建立情感連結的關鍵語句:

**開場共感:**
- 你也這樣想過嗎？
- 我懂你的心情
- 這個困擾很多人都有
- 你一定也遇過這種狀況

**中段共感:**
- 對吧？
- 是不是？
- 有沒有很熟悉的感覺？
- 你可能也注意到了

**結尾共感:**
- 我們一起努力
- 你準備好了嗎？
- 試試看吧
- 期待你的好消息

### 3. 柔性表達 (Soft Expression)

台灣文化偏好委婉、柔和的表達:

| 情境 | 硬性表達 ❌ | 柔性表達 ✅ |
|------|------------|------------|
| 否定 | 這是錯的 | 這個想法可能需要調整 |
| 建議 | 你必須這樣做 | 你可以試試這個方法 |
| 批評 | 你搞錯了 | 我有不同的看法 |
| 強調 | 絕對是這樣 | 我傾向認為 |

### 4. 情感助詞運用

適當使用情感助詞增加溫度:

**語氣詞:**
- 呢 (溫和疑問): 這樣好嗎呢？
- 喔 (理解回應): 原來是這樣喔
- 啦 (輕鬆肯定): 就這麼做啦
- 吧 (邀請同意): 一起試試吧

**加強詞:**
- 超 (程度加強): 超棒的、超實用
- 真的 (強調真實): 真的很重要
- 根本 (徹底否定): 根本不用擔心
- 完全 (全面肯定): 完全可以做到

## 句型結構建議

### 1. 段落長度

**台灣手機閱讀優先:**

❌ **太長** (一段7-8行):
```
當你40歲了還在問自己是不是太老這個問題的時候,
你需要知道的是,根據MIT、哈佛、Wharton的聯合研究,
成功創業者的平均年齡其實是45歲,而且50歲創業者的
成功率是30歲創業者的1.8倍,這些數據都清楚地表明,
年齡不僅不是障礙,反而是優勢。
```

✅ **適中** (一段2-3行):
```
你也40歲了,還在問自己會不會太老？

先看一個數據:
MIT研究發現,成功創業者平均45歲。

50歲創業成功率是30歲的1.8倍。

所以,年齡不是障礙,是優勢。
```

### 2. 開場句型

**吸引注意的台灣式開場:**

**類型1: 直接提問**
```
你也遇過這種狀況嗎？
每次想開始副業,第一個念頭就是...
```

**類型2: 情境描述**
```
半夜躺在床上滑手機,
看到朋友創業成功的貼文,
你心裡想:「我是不是錯過了？」
```

**類型3: 顛覆認知**
```
等等,先別急著否定自己。
我要告訴你一個反直覺的事實...
```

**類型4: 共感開場**
```
我懂你的擔心。
40歲了,家裡有房貸、有孩子,
怎麼敢冒險創業？
```

### 3. 過渡句型

**自然流暢的台灣式過渡:**

**轉折:**
- 但是,事情沒那麼簡單
- 問題來了
- 重點在這裡
- 先別急

**遞進:**
- 不只如此
- 更重要的是
- 還有一點
- 你可能沒想到

**舉例:**
- 舉個例子
- 像是
- 比方說
- 讓我說個故事

### 4. 結尾句型

**行動導向的台灣式結尾:**

**類型1: 溫和邀請**
```
準備好了嗎？
一起試試看吧。
```

**類型2: 具體挑戰**
```
3個月後,回來告訴我結果。
我等你的好消息。
```

**類型3: 留言互動**
```
你也40歲了嗎？
留言跟我分享你的想法。
```

**類型4: 鼓勵支持**
```
相信我,你可以的。
我們一起加油！
```

## Before/After 範例

### 範例 1: 開場優化

**Before (太正式,大陸風格):**
```
根據相關研究顯示,當今社會中許多40歲人士對於
創業存在較大疑慮,主要擔憂在於年齡因素可能對
創業成功率產生負面影響。然而,實際數據表明,
此種擔憂並無充分依據。
```

**After (台灣親切風格):**
```
你也40歲了,每次想到創業就開始焦慮？

我懂那種感覺。
「會不會太老了？」
「還來得及嗎？」
「失敗了怎麼辦？」

但數據告訴我們一個驚人的事實...
```

**改善重點:**
- ✅ 用「你」建立對話感
- ✅ 短句增加呼吸感
- ✅ 列出讀者內心疑問 (共感)
- ✅ 製造懸念引導閱讀

---

### 範例 2: 數據呈現優化

**Before (太學術):**
```
根據麻省理工學院、西北大學、沃頓商學院及
美國人口普查局針對270萬家企業創辦人的聯合
研究顯示,成功創業者的平均年齡為45歲,
前0.1%高成長企業創辦人平均年齡同為45歲。
```

**After (台灣易讀風格):**
```
先給你看一個數據,你可能會嚇一跳:

MIT、哈佛、Wharton 研究了270萬家公司發現——

✨ 成功創業者平均年齡:45歲
✨ 頂尖0.1%企業創辦人:也是45歲

沒錯,是45歲,不是25歲。
```

**改善重點:**
- ✅ 鋪墊情緒 (「你可能會嚇一跳」)
- ✅ 簡化機構名稱
- ✅ 視覺化呈現 (emoji + 換行)
- ✅ 重複強調驚訝點

### 範例 3: 行動呼籲優化

**Before (太命令式):**

**After (台灣邀請式):**

最後,給你一個3個月的小挑戰:

從今天開始——

3個月後,回來留言告訴我:

你願意試試看嗎? 💪
```

**改善重點:**
- ✅ 溫和邀請取代命令
- ✅ 視覺對比 (❌ vs ✅)
- ✅ 個人化互動 (用名字)
- ✅ 輕鬆問句結尾

## 台灣直接溝通風格

台灣讀者重視真誠、自然的溝通風格,對過度討好、吹捧或不自然的稱讚會產生排斥感。這個章節教你如何建立真誠、直接但仍保有溫度的溝通方式。

### 核心原則

**台灣直接溝通的三大支柱:**

1. **真誠優於討好** - 說真心話,不說場面話
2. **實用勝過漂亮話** - 給建議,不給空泛讚美
3. **對話感取代推銷感** - 像朋友聊天,不是業務話術

### 避免討好和吹捧

#### 過度讚美 (避免使用)

❌ **什麼是討好式語言:**

```
您真是太聰明了!
這個想法簡直天才!
您絕對是最棒的!
您的選擇非常英明!
```

**為什麼台灣讀者反感:**
- 感覺虛假、不真誠
- 像推銷話術
- 缺乏實質內容
- 過度恭維顯得不專業

✅ **改為真誠直接:**

```
這個想法有潛力,我們來討論一下。
你問到重點了,讓我跟你說...
我理解你的顧慮,這很正常。
這個方向可以試試看。
```

#### 對比範例

**情境 1: 回應使用者問題**

❌ **討好式:**
```
您提出的問題非常有深度!
您的思考能力真是令人佩服!
您一定是這個領域的專家!
```

✅ **直接真誠:**
```
你問到核心問題了。
這個問題很多人都會碰到,讓我跟你說我的想法。
我懂你為什麼會這樣想。
```

**情境 2: 給予建議**

❌ **討好式:**
```
以您的聰明才智,一定能做到最好!
您絕對會成功的,我完全相信您!
您太優秀了,根本不用擔心!
```

✅ **直接真誠:**
```
這個方法值得試試看。
我覺得你可以從這裡開始。
有困難很正常,慢慢來就好。
```

**情境 3: 鼓勵使用者**

❌ **討好式:**
```
您實在太棒了!
您是我見過最厲害的!
您的成功是必然的!
```

✅ **直接真誠:**
```
你做得不錯,繼續加油。
這個進度很好,保持下去。
我看好你,但還是要腳踏實地。
```

### 自然對話語氣

**台灣直接溝通的語氣特徵:**

#### 1. 用「我」表達觀點

✅ **主觀誠實:**
```
我的想法是...
我覺得...
我建議你可以...
依我的經驗...
```

❌ **過度客觀化:**
```
眾所周知...
毫無疑問...
絕對正確的是...
事實證明...
```

#### 2. 承認局限性

✅ **謙遜真實:**
```
這只是我的看法,你可以參考看看。
我不確定這對你適不適用。
這方法不一定對每個人都有效。
我也還在學習中。
```

❌ **過度自信:**
```
這絕對是唯一的答案。
你一定要這樣做。
不這樣做就是錯的。
我保證100%有效。
```

#### 3. 用問句創造對話

✅ **互動式表達:**
```
你覺得呢?
這樣說有道理嗎?
你也有類似的經驗吧?
我這樣解釋清楚嗎?
```

❌ **單向灌輸:**
```
你必須知道...
你應該要...
你一定要記住...
```

### 實用勝過空談

#### Before/After 對比

**案例 1: 文章開場**

❌ **空泛讚美開場:**
```
親愛的讀者,您今天又變得更優秀了!
感謝您願意花時間閱讀這篇文章,
您對自我提升的追求真是令人敬佩!
現在讓我們一起來學習這個主題吧!
```

✅ **直接進入主題:**
```
40歲開始副業,你最擔心什麼?

我收到最多的問題是:
「會不會太晚了?」

讓我直接告訴你答案...
```

**案例 2: 給建議**

❌ **包裝式空話:**
```
您的問題問得太好了!
這充分展現了您深度思考的能力!
以您這樣的洞察力,相信您很快就能找到答案!
我相信您一定可以做到的!
```

✅ **直接給實用建議:**
```
你問到重點了。

我建議你這樣做:
1. 先用3個月測試市場反應
2. 找5-10個潛在客戶談談
3. 記錄他們的真實反饋

這樣你就會知道方向對不對。
```

**案例 3: 回應留言**

❌ **社交辭令式:**
```
非常感謝您寶貴的留言!
您的見解真是太精闢了!
期待您繼續分享您的智慧!
```

✅ **真誠互動:**
```
謝謝你的分享。
你提到的這個點我也遇過。
我們可以再聊聊這個。
```

### 建立信任的直接溝通

**信任來自真誠,不是吹捧。**

#### 信任建立公式

✅ **台灣風格信任溝通:**

1. **承認不完美**
```
我也不是專家,但我可以跟你分享我的經驗。
這個方法對我有用,但不保證對每個人都適合。
我也還在學習,我們一起進步。
```

2. **提供具體價值**
```
這是我實際測試過的3個方法。
我花了2年時間學到的5個教訓。
這些數據來自我訪談的50位創業者。
```

3. **邀請對話而非說教**
```
你的狀況是什麼?我們可以討論看看。
如果你試過了,歡迎回來跟我分享結果。
有問題隨時問,我能幫的會盡量幫。
```

### 檢查清單

使用這個清單確保你的溝通風格符合台灣直接真誠標準:

**避免討好檢查:**
- [ ] 沒有過度讚美 (您真是太XXX了)
- [ ] 沒有空泛恭維 (您一定會成功的)
- [ ] 沒有推銷式吹捧 (絕對、保證、100%)
- [ ] 沒有不真誠的客套話

**真誠直接檢查:**
- [ ] 使用第一人稱表達觀點 (我覺得、我建議)
- [ ] 承認局限性和不確定性
- [ ] 提供具體實用建議而非空話
- [ ] 創造對話感而非單向說教

**語氣平衡檢查:**
- [ ] 直接但不粗魯
- [ ] 真誠但不冷漠
- [ ] 實用但有溫度
- [ ] 專業但親和

## 謙遜不武斷語氣

台灣文化重視謙遜、不過度自信的表達方式。這個章節教你如何在保持專業性的同時,使用柔和、開放的語氣,避免過度武斷或絕對化的表達。

### 核心理念

**謙遜語氣的價值:**
- 給讀者思考空間
- 降低防禦心理
- 建立平等對話感
- 展現文化素養

**不是要你:**
- 失去專業性
- 缺乏自信
- 模糊立場
- 迴避責任

**而是要你:**
- 留有餘地
- 尊重差異
- 開放討論
- 理性表達

### 柔化絕對化表達

#### 避免絕對化語言

❌ **過度武斷的表達:**

```
這絕對是唯一的方法
你一定要這樣做
這100%有效
沒有例外
所有人都應該...
永遠不要...
```

✅ **謙遜開放的表達:**

```
這可能是比較適合的方法
你可以考慮這樣做
這在大多數情況下有效
通常來說...
很多人會...
盡量避免...
```

#### 對比範例

**情境 1: 給建議**

❌ **武斷式:**
```
你必須立刻開始行動!
不這樣做就是錯的!
沒有其他選擇!
```

✅ **謙遜式:**
```
我建議你可以開始試試看。
這個方向可能比較適合你。
當然,你也可以考慮其他選項。
```

**情境 2: 分享經驗**

❌ **武斷式:**
```
這絕對是最好的方法!
我保證有效!
任何人都能成功!
```

✅ **謙遜式:**
```
這個方法對我有用,你可以參考看看。
我覺得有機會,但需要測試。
有些人可能會成功,但不保證適合所有人。
```

**情境 3: 表達觀點**

❌ **武斷式:**
```
事實就是如此!
毫無疑問...
所有專家都同意...
```

✅ **謙遜式:**
```
依我的理解...
我傾向認為...
許多專家認為...
```

### 謙遜用詞選擇

#### 高頻謙遜詞彙

**程度柔化詞:**

| 武斷 ❌ | 謙遜 ✅ |
|---------|---------|
| 一定 | 可能、大概、也許 |
| 絕對 | 通常、往往、多半 |
| 必須 | 建議、可以、不妨 |
| 肯定 | 應該、或許、傾向 |
| 永遠 | 常常、經常、大多數時候 |
| 所有 | 很多、大部分、多數 |

**建議表達詞:**

| 強硬 ❌ | 柔和 ✅ |
|---------|---------|
| 你必須 | 你可以考慮 |
| 你應該 | 我建議你 |
| 你要 | 不妨試試 |
| 不要 | 盡量避免 |
| 千萬別 | 最好不要 |

**觀點表達詞:**

| 武斷 ❌ | 謙遜 ✅ |
|---------|---------|
| 事實是 | 依我的理解 |
| 毫無疑問 | 我傾向認為 |
| 顯然 | 看起來似乎 |
| 明顯 | 可能是因為 |
| 必然 | 通常會 |

### 開放式結尾

#### 邀請對話,不下結論

✅ **開放式結尾模板:**

**類型 1: 徵詢意見**
```
你覺得呢?
你有不同的看法嗎?
我這樣說有道理嗎?
你的狀況可能不太一樣,歡迎分享。
```

**類型 2: 承認多元性**
```
這只是我的經驗,不一定適合每個人。
當然,每個人的狀況都不同。
你可能有更適合自己的方法。
這個建議僅供參考。
```

**類型 3: 邀請討論**
```
如果你有不同經驗,歡迎分享。
我很想知道你的想法。
我們可以一起討論看看。
有其他角度也可以提出來。
```

**類型 4: 持續開放**
```
我也還在學習中。
這個問題沒有標準答案。
我們都在摸索更好的方法。
歡迎一起交流。
```

### 保持專業性的謙遜

**重要:** 謙遜不等於缺乏自信或專業性不足。

#### 平衡點

✅ **專業 + 謙遜:**

```
依我10年的實務經驗,這個方法在大多數情況下是有效的。
當然,每個產業的狀況不同,你需要根據自己的情況調整。
如果你在實踐過程中遇到問題,歡迎隨時討論。
```

**分析:**
- ✅ 建立專業性 (10年經驗)
- ✅ 柔化表達 (大多數情況)
- ✅ 承認差異性 (每個產業不同)
- ✅ 開放對話 (歡迎討論)

❌ **過度謙遜 (失去專業性):**

```
我也不太懂啦,只是隨便說說。
這可能完全不對,你別信我。
我什麼都不知道,你問別人吧。
```

**問題:**
- ❌ 過度自我貶低
- ❌ 失去可信度
- ❌ 沒有提供價值

### Before/After 完整範例

**範例 1: 專業建議文章**

❌ **過度武斷版:**
```
40歲創業絕對來得及!

這是毫無疑問的事實。
所有數據都證明這一點。
你必須立刻開始行動。
不行動就是在浪費時間!
相信我,這絕對是正確的決定!
你一定會成功的!
```

✅ **謙遜專業版:**
```
40歲創業,我覺得還來得及。

從數據來看,很多成功創業者都是40歲以上開始的。
如果你有這個想法,我建議可以試試看。
當然,每個人的狀況不同,你需要評估自己的條件。
這只是我的看法,你可以參考看看。
如果決定嘗試,我們可以一起討論怎麼開始。
```

**範例 2: 回應讀者問題**

❌ **武斷版:**
```
你的想法完全錯誤!
這絕對不可能成功!
你必須改變思維方式!
聽我的就對了!
```

✅ **謙遜版:**
```
我理解你的想法,但我有些不同的看法。
依我的經驗,這個方向可能會比較困難。
我建議你可以考慮從另一個角度切入。
當然,這只是我的建議,最終還是要看你的判斷。
```

### 文化脈絡

**為什麼台灣重視謙遜語氣:**

1. **儒家文化影響** - 謙虛是美德
2. **避免衝突** - 和諧溝通優先
3. **尊重多元** - 承認不同可能性
4. **建立信任** - 真誠勝過自大

**與其他文化對比:**

| 文化 | 表達風格 | 範例 |
|------|---------|------|
| 台灣 | 謙遜開放 | 我覺得、可能、建議你可以 |
| 美國 | 直接自信 | You should、This is the best |
| 大陸 | 權威論述 | 必須、應當、正確的做法是 |

### 檢查清單

**武斷語言檢查 (避免):**
- [ ] 絕對、一定、必須、肯定
- [ ] 100%、永遠、所有、每個
- [ ] 毫無疑問、顯然、明顯
- [ ] 你一定要、你必須、不這樣就...

**謙遜語言檢查 (使用):**
- [ ] 可能、也許、大概、或許
- [ ] 我覺得、我建議、依我看
- [ ] 通常、大多數、很多時候
- [ ] 你可以考慮、不妨試試、建議

**開放態度檢查:**
- [ ] 承認局限性和不確定性
- [ ] 尊重不同觀點和情況
- [ ] 邀請對話而非下結論
- [ ] 留有討論和調整空間

**專業平衡檢查:**
- [ ] 保持專業可信度
- [ ] 提供具體價值和建議
- [ ] 謙遜但不自我貶低
- [ ] 柔和但不模糊立場

## 格式風格偏好

台灣讀者(特別是手機閱讀為主的社群媒體使用者)偏好自然流暢的段落式閱讀,而不是過度結構化的markdown格式。這個章節教你如何平衡可讀性與格式化,創造舒適的閱讀體驗。

### 核心理念

**台灣閱讀偏好:**
- 自然段落 > 項目符號列表
- 對話感 > 簡報感
- 流暢閱讀 > 資訊切割
- 故事敘事 > 條列重點

**但不是完全避免列表,而是:**
- 策略性使用列表
- 列表與段落平衡
- 根據內容類型選擇格式
- 優先考慮閱讀體驗

### 段落 vs 列表的使用時機

#### 優先使用自然段落

✅ **適合段落的內容類型:**

**1. 故事和經驗分享**
```
我記得剛開始創業的時候,每天半夜兩點還在改企劃書。
那時候真的很累,常常懷疑自己是不是做錯決定。
但現在回頭看,那段時間的掙扎反而讓我成長最多。
你現在的焦慮,我都經歷過,我懂。
```

**2. 情感共鳴內容**
```
你也40歲了,還在猶豫要不要開始副業？

我懂那種焦慮感。每次滑到朋友創業成功的貼文,
心裡就會想:「我是不是錯過了?」

這種感覺很正常,我也有過。
但讓我告訴你一個你可能不知道的事實...
```

**3. 深度分析和論述**
```
為什麼40歲創業反而更有優勢?

這不是安慰你,而是有數據支撐的。MIT研究發現,
成功創業者的平均年齡是45歲,不是25歲。原因很簡單:
經驗、人脈、資源,這些都需要時間累積。

年輕人有熱情,但中年人有智慧。
你現在擁有的,是20歲的你拿不到的武器。
```

#### 策略性使用列表

✅ **適合列表的內容類型:**

**1. 具體步驟和流程**
```
如果你決定開始,我建議這樣做:

1. 先用3個月測試市場反應
2. 找5-10個潛在客戶深度訪談
3. 記錄他們的真實需求和願付價格
4. 根據反饋調整你的服務內容
5. 有信心了再擴大規模

這5個步驟能幫你降低風險。
```

**2. 對比和選項**
```
40歲創業有兩種路線:

**路線A: 全職創業**

## 可讀性語言層級

台灣內容創作的關鍵成功因素之一是「好讀」— 讓不同教育背景的讀者都能輕鬆理解。這個章節教你如何使用國中生能理解的語言,在保持專業性的同時提升可讀性。

### 核心理念

**可讀性金字塔:**

```
專家級語言 (學術論文)
        ↓
專業級語言 (產業報告)
        ↓
【國中生語言】← 目標層級
        ↓
小學生語言 (過度簡化)
```

**為什麼選擇國中生語言層級:**

✓ **覆蓋率最高** - 99%台灣人都能理解
✓ **理解最快** - 不需要反覆閱讀
✓ **分享率高** - 容易轉發給他人
✓ **專業與親和平衡** - 不失專業性又易懂

### 簡化原則

#### 原則 1: 用白話替換專業術語

**轉換公式:**

```
專業術語 → 白話解釋 + (專業術語)
```

**範例:**

❌ **過度專業:**
```
我們需要進行市場區隔 (Market Segmentation),
針對目標客群 (Target Audience) 進行價值主張 (Value Proposition) 的傳遞,
以提升轉換率 (Conversion Rate)。
```

✅ **國中生能懂:**
```
我們要先搞清楚:你的客戶是誰?他們要什麼?

找對人,說對話,客戶才會買單。

(行銷術語叫這個「市場區隔」和「價值主張」,
但你只要記住:找對人,說對話就好。)
```

**常見專業術語的白話版:**

| 專業術語 | 國中生版本 |
|---------|-----------|
| 市場區隔 | 找出你的目標客戶是誰 |
| 價值主張 | 告訴客戶你能幫他什麼 |
| 轉換率 | 有多少人看了會買 |
| ROI 投資報酬率 | 花1塊賺回多少 |
| 客戶痛點 | 客戶最頭痛的問題 |
| 競爭優勢 | 你比別人強在哪 |
| 商業模式 | 你怎麼賺錢 |
| 現金流 | 戶頭裡有多少現金可以用 |

#### 原則 2: 拆解複雜句

**複雜句 → 短句群**
