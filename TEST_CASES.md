## 快速測試指南

### 測試目的

1. **驗證 Agents 設置正確**: 每個 Agent 的 System Prompt 正常工作
2. **測試工作流程順暢**: 資訊能在 Agents 間正確傳遞
3. **評估輸出品質**: 生成的內容符合預期
4. **發現優化空間**: 找出需要調整的地方

### 測試建議

**Day 1 - 核心功能測試**:
- 測試案例 1 (SOLVE Method)
- 驗證 Phase 1-3 的基本流程

**Day 2 - 擴展功能測試**:
- 測試案例 2 (BUSINESS Method)
- 測試案例 3 (HEART Method)
- 驗證完整 4 Phase 流程

**Day 3 - 完整系統測試**:
- 測試案例 4 和 5
- 嘗試自己的真實案例
- 優化 System Prompts

## 測試案例 1: 專業教學內容 (SOLVE)

### 案例背景

**場景**: 你是一位數位行銷專家,想分享如何提升網站 SEO 排名的實用方法
**目標**: 提供actionable的教學指南
**受眾**: 中小企業主、行銷新手
**平台**: Blog / LinkedIn
**預期框架**: SOLVE Method

---

### Phase 1 測試腳本

#### Test 1.1: Master Workflow Agent

**輸入**:
```
我想創作一篇關於 SEO 優化的教學內容。

【個人背景】
- 職業: 數位行銷顧問
- 經驗: 5 年 SEO 和內容行銷經驗
- 專業: 幫助中小企業提升網站流量和排名

【創作需求】
- 主題: 如何在 3 個月內提升網站 SEO 排名
- 目標: 提供實用、可執行的指南,建立專業權威
- 目標受眾: 中小企業主、行銷新手
- 主要平台: Blog 和 LinkedIn

【個人特質】
- 寫作風格: 實務導向,重視可操作性
- 擅長: step-by-step 指導,用實際案例說明
- 個性: 理性邏輯型,喜歡用數據說話

請幫我分析並推薦最適合的內容創作框架。
```

**預期輸出檢查**:
- [ ] Agent 識別出"實務導向型"特質
- [ ] Agent 識別出"知識分享"或"建立權威"目標
- [ ] Agent 推薦 SOLVE, TEACH, 或 LEAD 方向
- [ ] 提供清晰的下一步建議

---

#### Test 1.2: Framework Selector Agent

**輸入**:
```
[背景資訊]
- 用戶特質: 實務導向型,理性邏輯型
- 內容目標: 知識分享,建立權威
- 目標平台: Blog 和 LinkedIn
- 目標受眾: 中小企業主、行銷新手

[具體需求]
- 主題: 如何在 3 個月內提升網站 SEO 排名
- 個人風格: 實務導向,step-by-step 指導,數據支撐
- 期望效果: 讀者能夠立即應用並看到效果

請為我推薦前 3 名最適合的寫作框架,並說明理由和預期效果。
```

**預期輸出檢查**:
- [ ] 推薦前 3 名框架
- [ ] SOLVE Method 應該在前 2 名(匹配度 85+)
- [ ] 每個框架有明確的匹配分數
- [ ] 提供選擇理由和預期效果
- [ ] 包含實用的使用技巧

**關鍵資訊記錄**:
- 選定框架: ________________
- 匹配分數: ________________
- 核心理由: ________________

### Phase 2 測試腳本

#### Test 2.1: Interview Specialist Agent

**輸入**:
```
[背景資訊]
- 選定框架: SOLVE Method
- 內容主題: 如何在 3 個月內提升網站 SEO 排名
- 目標受眾: 中小企業主、行銷新手
- 創作目標: 提供實用 actionable 的指南

[我的經驗背景]
我有 5 年的 SEO 和內容行銷經驗,專門幫助中小企業提升網站流量和排名。我曾經幫助一家本地咖啡店在 3 個月內將自然流量提升了 300%,Google 排名從第 3 頁提升到第 1 頁前 5 名。我的方法重視實用性和可執行性,不需要大預算或技術背景。

請開始訪談我,幫我挖掘出創作這篇內容所需的素材。
```

**訪談互動測試**:

模擬回應以下問題:

**Q1 (暖場)**: "請簡單介紹一下你和咖啡店客戶合作的背景"

**A1 (你的回應)**:
```
這是一家開業 2 年的本地咖啡店,老闆娘很有心經營,但網站幾乎沒有流量。他們的網站是用 WordPress 套模板做的,很陽春,也沒有任何 SEO 優化。老闆娘找到我時很焦慮,因為周圍開了很多連鎖咖啡店,她擔心生意被搶走。
```

**Q2 (核心事實)**: "你具體做了哪些優化工作?"

**A2 (你的回應)**:
```
我分成三個階段執行:

第一個月:
- 技術SEO修復:改善網站速度(從4.5秒降到1.2秒)
- 建立 Google My Business 並優化
- 關鍵詞研究,鎖定"XX區咖啡店"等本地關鍵詞

第二個月:
- 創建 15 篇部落格文章(咖啡知識、店內活動、本地生活)
- 優化所有頁面的 meta tags 和 alt texts
- 建立內部連結結構

第三個月:
- 獲得 10 個本地媒體和部落格的反向連結
- 發布客戶評價和案例
- 持續內容更新
```

**Q3 (深度洞察)**: "過程中遇到最大的挑戰是什麼?"

**A3 (你的回應)**:
```
最大的挑戰是讓老闆娘理解 SEO 需要時間。她一開始期待一週內就看到結果,當第一個月流量沒有明顯變化時,她非常焦慮,甚至想放棄。

我用了兩個方法說服她:
1. 每週用 Google Search Console 數據展示進步(雖然微小但是在進步)
2. 分享其他類似案例的時間線

第二個月開始看到明顯效果後,她就變得很配合了。
```

**Q4 (情感時刻)**: "最讓你有成就感的時刻是什麼?"

**A4 (你的回應)**:
```
第三個月結束時,老闆娘打給我說"我們今天有 5 個客人說是從 Google 搜尋找到我們的!"她的聲音充滿興奮和感激。

那一刻我覺得一切努力都值得了。不只是數字的提升,而是真正幫助一個小企業主實現了她的夢想,讓她能夠和連鎖店競爭。
```

**Q5 (未來展望)**: "基於這個經驗,你對想做 SEO 的中小企業主有什麼建議?"

**A5 (你的回應)**:
```
三個核心建議:

1. 耐心是關鍵:SEO 需要 3-6 個月才能看到明顯效果,不要期待速效
2. 從本地 SEO 開始:對中小企業來說,本地排名比全國排名更實際
3. 內容是王道:不需要高超技術,但需要持續產出有價值的內容

最重要的是:開始行動比完美計劃更重要。很多人因為害怕做錯而遲遲不開始,但 SEO 是可以邊做邊調整的。
```

**預期輸出檢查**:
- [ ] 完成 5 階段訪談 (暖場 → 事實 → 洞察 → 情感 → 展望)
- [ ] 獲得結構化的素材整理
- [ ] 素材包含:核心故事線、關鍵洞察、情感時刻、實用建議、金句、數據
- [ ] 素材適合 SOLVE Method 框架
